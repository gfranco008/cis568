Why This Page Matters for the Thesis

Your core question is not only trajectory behavior at a fixed μ, but also how accurately we can predict across μ from a limited set of training parameter samples. This page fixes time t* and treats the state as a function of μ, then builds barycentric surrogates from N training nodes.

By comparing equispaced vs Chebyshev μ nodes, you can see where node placement changes the error profile over the parameter domain. The convergence panel quantifies error decay as N increases, and the runtime panel shows computational cost, so the method can be judged on both accuracy and efficiency.

Research significance: this is the operational view used in your surrogate workflow: select μ training points, interpolate parametric ODE outputs, and evaluate error/cost tradeoffs before downstream tasks such as emulator design and model comparison.
